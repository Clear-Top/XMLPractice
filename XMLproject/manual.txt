#Load
1. 현재디렉토리 파일의 XML 파일을 선택합니다.
2. 로딩이 완료되면, 초기화면에 빨간글씨로 표시됩니다.
(단, 이미 로딩된파일이 있다면 로딩사용이 불가합니다.)
#End

#Make
- DOM Tree를 처음부터 작성합니다.
(단, 이미 로딩된 파일이 있다면 저장후 작업이 가능합니다.)
#End

#Find
1. 원하는 노드의 타입을 지정합니다.
2. 원하는 노드의 이름을 지정합니다.
3. 생성된 버튼을 클릭하면, 정보를 볼 수 있습니다.
(단, 동일한 이름이 여러개라면 모두 조회)
#End

#Save
1. 변환할 파일이름을 입력합니다.
2. 현재작업디렉토리를 확인합니다.
(단, 동일한 이름의 파일이 존재할경우 덮어씁니다.)
#End

#Print
1. 현재작업중인 DOM Tree를 텍스트 형태로 확인합니다.
2. 확인이 끝나면 "확인"버튼을 눌러 종료합니다.
#End

#Insert
1. 조회버튼을 클릭합니다.
2-1. 우클릭을 통해 선택한 노드로 진입합니다.
2-2. 좌클릭을 통해 선택한 노드"앞"에 노드를 삽입합니다.
3. "부모로 이동"을 누르면 뒤로 갈 수 있습니다.
4. "초기화" 버튼을 클릭하면 모든 설정이 초기화됩니다.
#End

#Update
this is Update button
#End

#Delete
1. 조회버튼을 클릭합니다.
2. 좌클릭을 통해 선택한 노드를 삭제합니다.
3. "부모로 이동"을 누르면 뒤로 갈 수 있습니다.
4. "초기화" 버튼을 클릭하면 모든 설정이 초기화됩니다.
#End

#Exit
1. 이미 로딩된 파일이 있다면 XML파일로 저장후 종료가능합니다.
2. 종료합니다.
#End
